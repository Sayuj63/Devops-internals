Runbook

Eight alerts. Eight playbooks.

The on-call rotation gets paged for exactly these alerts. Each entry lists what it means, the diagnostic queries you run first, and the safe remediation step. Anything outside this list is a Sev-3 ticket, not a page.

CriticalAPIHighErrorRate
Meaning
5xx rate on /api/v1/sims/* above 2% for 5 minutes. Customer-visible.
Diagnose
sum(rate(http_requests_total{job="api",status=~"5.."}[5m])) / sum(rate(http_requests_total{job="api"}[5m]))
Kibana
kubernetes.labels.app:"api" AND status:[500 TO 599] — group by trace_id.
Remediate
If a single trace dominates, identify the downstream failure (Vault, RDS, HLR). If broad, roll back to previous tag: kustomize edit set image api=ghcr.io/itm-sayuj/sim-ops/api:<prev> && kubectl apply -k k8s/overlays/prod.
CriticalSLOFastBurn
Meaning
14.4× burn rate over a 1-hour window. At this pace the monthly error budget is exhausted in two days.
Diagnose
Open the SLO board in Grafana — check whether the burn is from latency, errors, or both.
Remediate
Freeze deploys via scripts/freeze.sh until burn rate < 1.0 for 30 min. Investigate top-N endpoints by error contribution.
HighSimActivationStuck
Meaning
Any SIM has stayed in ALLOCATED for >10 minutes — worker is not draining the queue.
Diagnose
sim_state_age_seconds{state="ALLOCATED"} > 600 · worker queue depth: sim_worker_queue_depth
Remediate
If queue depth is climbing, scale worker: kubectl scale deploy/worker --replicas=4 -n sim-ops. Otherwise inspect the HLR sidecar — a stuck downstream usually blocks the state machine.
HighPostgresConnHigh
Meaning
Postgres connection count above 80% of max_connections. Activation traffic will start failing soon.
Diagnose
pg_stat_database_numbackends / pg_settings_max_connections > 0.8
Remediate
Check pgbouncer pool stats: kubectl exec -it pgbouncer-0 -- psql -p 6432 pgbouncer -c "SHOW POOLS". If a pod is leaking connections, restart it; otherwise raise max_connections via parameter group and apply.
HighVaultLeaderLost
Meaning
Vault has no active leader for >60s. Pods will fail to renew DB credentials.
Diagnose
vault_core_active == 0 on all peers. Run vault operator raft list-peers.
Remediate
Follow DR-03 in dr-plan.html. Quorum loss requires snapshot restore.
HighPodCrashLooping
Meaning
A pod has restarted >5 times in 15 minutes.
Diagnose
increase(kube_pod_container_status_restarts_total[15m]) > 5
Remediate
kubectl describe pod <name> -n sim-ops · kubectl logs --previous <name>. If config-related, fix and redeploy. If image-related, roll back.
MediumDiskPressureNode
Meaning
An EKS node's ephemeral storage usage exceeds 85%. Kubelet will start evicting soon.
Diagnose
node_filesystem_avail_bytes / node_filesystem_size_bytes < 0.15
Remediate
Identify largest containers via kubectl top pods. Likely a runaway log file — bounce the workload and confirm Filebeat is shipping.
LowCertExpiry30d
Meaning
Any managed certificate is within 30 days of expiry.
Diagnose
(certmanager_certificate_expiration_timestamp_seconds - time()) / 86400 < 30
Remediate
cert-manager renews automatically. If renewal failed, force it: kubectl annotate certificate sim-ops-tls cert-manager.io/force-renew=true. Confirm new secret reaches the ingress with openssl s_client -connect sim-ops.itm.edu:443 -servername sim-ops.itm.edu.
